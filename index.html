🥇 Ranking Rules
1

Points

Team with higher points ranks higher. Win more, rank higher.

2

Net Run Rate (NRR) ⭐ Tiebreaker

If points are equal, NRR decides the rank. Higher NRR = higher rank.

🧮 How NRR Works
Net Run Rate Formula
NRR  =  ( Runs Scored ÷ Overs Faced )
         −  ( Runs Conceded ÷ Overs Bowled )
📌 Example — Team A vs Team B
Team A bats
Scores 60 runs in 6 overs
Run Rate = 60 ÷ 6 = 10.00
Team B bats
Scores 50 runs in 6 overs
Run Rate = 50 ÷ 6 = 8.33
✅ Team A's NRR = 10.00 − 8.33 = +1.67  (Better ranking)
